## Install

[Download the lastest version](https://github.com/FulltronService/Ionic-Creator/releases).

## Development

- Clone: `$ git clone https://github.com/FulltronService/Ionic-Creator.git`
- Install NPM: `$ npm install`
- Start Preview: `$ npm start`
- Build: `$ npm run dist`
